Theme Customizer

You can customize the website using customizer options Click on Appearance > Customizer > Knowing God Customizer, and you will see the theme options. Set the options according to your needs. The theme options are listed below with explanations.

  1. Site Identity - Site logo, Site title , Site icon options are here.
  2. Knowing God Customizer - Manage blog post settings, Header settings-banner, breadcrumb , Footer and social Icons settings.
  3. Colors - Manage background color settings.
  4. Header Image - Customize your website header image
  5. Background Image - Manage background settings.
  6. Menus - Manage menu settings, display location
  7. Widgets - All sidebar and footer theme related widgets can be found here.
  8. Home Page Settings - Display of Front page and blog page settings
