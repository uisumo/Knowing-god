KNOWING GOD-CMS

How to Create Navigation menu?

  1. Go to Appearance->Menus
  2. Click on create New menu
  3. Add pages from the left side bar
  4. And Click on "save menu"

You can select the menu to be either footer menu or main navigation menu.

How to edit Sliders?

  1. Go to Slider Revolution settings.
  2. Select the text / layer you want to edit.
  3. You can find the options to easily change the font, color .
  4. And Click on "save".

-You have options to import and export the sliders.

-You have options to select the type of sliders you want.

How to create Blog Posts?

  1. Go to Posts.
  2. Create New Post.
  3. Select the categories or tags you want to allocate for the post.
  4. Select the post image.
  5. And Click on "publish".

How to create different page layouts?

  1. Once you create a page you will find the options to change the layout of the page
    • Default Page
    • Full Page
    • Revolution Full Page

    By default , the page will be full width without any side bar.

  2. You can choose page options for
    • Breadcrumb show/hide
    • Page Title Show/Hide
    • Side bar Postions- Left/ Right/ None.
  3. And Click on "publish".

How to edit Side bar Widgets?

  1. Go to Appearance-> Widgets.
  2. Drag and drop the widgets you want to the side bar / Footer.

You have options to edit the text in each widget.
