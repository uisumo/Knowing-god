1.Unit Testing

  • Posts display correctly, with no apparent visual problems or errors.
  • Posts display in correct order.
  • Correct number of posts display (as per setting in Settings > Reading).
  • Page navigation displays and works correctly.
  • No effect on the content with or without post featured images.
  • Categories/Tags display properly.
  • Sticky post display properly.
  • Images are aligned properly: Center, Left, Right, None.Image left, center and right alignment, horizontal and vertical alignments in post.
  • Text left, center and right alignment in post.
  • Floats are cleared properly for floated element .
  • The Blog Posts index page displays properly
  • The Search Results page displays properly, with search query results displayed.
  • Protected Test With Secret Password.
  • MORE info related to UNIT ESTING

2. Widget Testing

  • Widgets display correctly in all screen sizes.
  • Replacing the widgets results no harm - Worpress provides some default widgets which can be easily used/replaced by simple drag and drop..
  • Widgets includes - simple search, categories, dropdown search, calender etc;
Monster Widget is used for the widget testing.

MONSTER WIDGET:

- The Monster widget consolidates all 13 core widgets into a single widget enabling theme developers to create multiple instances with ease.
- It has been created to save time during theme development and review by minimizing the steps needed to populate a sidebar with widgets.
- The Monster widget is not designed for use in production.
